className和classList的那些事

发表于2022-11-21|更新于2022-11-21|学习笔记
|字数总计:1.6k|阅读时长:7分钟

写在前面

在制作手风琴效果时，会对一些元素的类名进行增加，修改，删除。可以使用classList和className进行控制，那么他们有什么区别呢？

以下的js代码均根据下方标签进行测试

className

概述

className 获取或设置指定元素的 class 属性的值。

cName是一个字符串变量，表示当前元素的class属性的值，可以是由空格分隔的多个class属性值。

拓展案例

注意

  • className是使用新值换旧值，如果需要添加一个类，就需要保留之前的类名
  • 可以同时指定多个样式类名，以空格分隔，空格的数量>=1

classList

概述

为了解决className容易覆盖以前的类名，我们可以通过classList的方式追加和删除类名。

Element.classList 是一个只读属性，返回一个元素 class 属性的动态 DOMTokenList 集合。这可以用于操作 class 集合。

用法

MDN案例

拓展案例

注意

  • classList.add()在追加一个类名时，“不影响”其他样式，若新添加的类有相同的样式时，会导致样式覆盖
  • classList.remove()在去除一个类名时，不影响其他样式
  • classList.toggle() 如果有该类名就移除，如果没有该类名就添加，名称只能存在一个，不能同时多个存在。若存在多个，只有第一个生效
  • classList.contains()会返回一个布尔值，如果参数里面同时有多个类名，不会报错。但是只会判断第一个类名，其他类名不会进行判断
  • classList.add()，classList.remove()在修改多个类名时，以，进行分隔
