Linux必知必会——常用命令

发表于2022-11-11|更新于2022-11-11|学习笔记
|字数总计:702|阅读时长:2分钟

前言

本文是Linux必知必会系列第二部分，Linux的常见命令。对命令行越熟练，使用Linux系统越得心应手。

前置知识

命令分类

内部命令 外部命令
属于Shell的一部分 属于Shell解释器程序之外的命令
没有单独对应的系统文件 每个外部命令对应了系统中的一个文件
自动载入内存，可以直接使用 必须知道其对应的文件位置，由Shell加载后才能执行

命令可以通过 type 来判断属于内部命令还是外部命令

文件类型

文件类型 ls -l指令结果（第一个字符）
普通文件 -
目录文件 d
设备文件 （b）块设备;（c）字符设备
链接文件 l
管道文件 p
套接字文件 s

命令集

帮助

help

help 命令用于查看Bash内部命令的帮助信息

man

man 命令主要用于获取帮助信息

目录操作

pwd

pwd 命令主要用于显示或者查看当前所在的目录绝对路径

cd

cd 命令主要用于目录切换。如 cd ../ 表示切换到上级目录； cd ./ 表示切换到当前目录；cd /home 表示以根目录为参考切换到 home 目录；cd ./dir 表示以当前目录为参考，切换到 dir 目录下。

注意

  • 绝对路径：从linux最底层的根 ( / ) 开始算，例如：/home,/root 就是绝对路径

  • 相对路径：相对当前你所在路径开始算，例如：./dir,dir 就是相对路径

ls

ls 命令主要用于浏览目录下的文件或者文件夹。

选项 功能
-l 查看详细信息，元数据信息(用户、组、大小、创建时间、权限信息、文件类型)
-a 显示隐藏文件（例如 .ssh）

如：ls ./ 查看当前目录下的文件列表

mkdir

mkdir 命令主要用于创建目录。

选项 功能
-p 递归创建

如 mkdir a b c 表示创建a，b，c三个文件夹；mkdir -p /a/b/c 表示在 a 目录下创建目录 b ，在 b 目录下创建目录 c

touch

touch 命令主要用于创建普通文件，如果文件已经存在则刷新文件的时间戳。

rm

rm 命令主要用于删除文件或者目录

选项 功能
-r 递归删除
-f 强制删除
-i 删除前确认

如 rm -rf aa/test.txt 表示强制删除 aa 目录下的 test.txt 文件

ln

ln 命令主要用于创建文件链接。链接分为两种方式：硬链接和软连接

区别：

对比项 硬链接 软连接
创建方式 选项加-s是软连接 选项不加-s是硬连接
本质 本质是同一个文件 本质不是同一个文件

cp

cp 命令主要用于复制文件或目录

选项 功能
-b 若存在同名文件，则目标目录中的文件创建副本（文件名~）
-i 复制前确认
-r 递归复制目录及其子目录内的所有内容

mv

mv 命令主要用于移动文件或目录，如果新旧路径不一样就是移动， 如果新旧路径相同就是改名（只能一个个改）。如果目标路径存在相同文件，则表示覆盖。

选项 功能
-b 若存在同名文件，则目标目录中的文件创建副本
-v 移动成功后显示详细操作信息

如 mv ./dir/old.txt ./dir/new.txt 表示将 dir 目录下的 old.txt 文件改为 new.txt

file

file 命令主要用于查看文件类型

cat

cat 命令主要用于查看文件内容

选项 功能
-n 为输出的所有行编号

如 cat test.txt 表示查看 test.txt 的内容

more

more 命令主要用于全屏方式显示文件内容

快捷键 功能
Enter 向下逐行滚动
空格 向下翻一屏
b 向上翻一屏
q 退出

less

less 命令主要用于分屏方式显示文件内容

快捷键 功能
/ word 文件中查找单词（光标之下）
? word 文件中查找单词（光标之上）

搜索与查找

which

which 命令会在环境变量设置的目录里查找符合条件的文件

find

find 命令用来从指定目录向下递归查找指定的文件

  • 查找范围：指定具体目标路径，默认为当前目录
  • 查找条件：可以对文件名、大小、类型、权限等标准进行查找，默认为找出指定路径下的所有文件
条件 说明
-type 根据文件类型
-size 根据文件的大小进行查找一般使用“＋”(大于)、“-”（小于）号作为查找条件常用的容量单位包括 kB（小写）、MB、GB
-name 根据目标文件的名称进行查找,允许使用“*”及“?”通配符必须加上" "
-user 查找给定用户的文件
  • 动作选项：对符合条件的文件做操作，默认输出至屏幕（print）
动作 说明
-print 打印找到的每个文件名
-exec command {} \; 对找到的文件执行command命令
-ok command {} \; 执行command前确认

如find data -type f -name "*.log" -exec rm -rf {} \; 表示找到data目录下以.log结尾的普通文件并删除

Linux命令行快捷键

快捷键 功能
tab 自动补齐命令

结语

linux命令可以结合使用，单个命令也可以配合不同的选项，要善用通配符，通配符需要加上英文模式下的双引号，linux都需要英文模式下输命令，以免影响操作
